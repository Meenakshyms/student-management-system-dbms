# DBMS Student Management System

A web-based Student Management System developed using Django to demonstrate DBMS concepts such as data storage, retrieval, and validation.

## Features
- Student registration
- Server-side form validation
- Secure form submission using CSRF protection
- Displays stored student records

## Technologies Used
- Python
- Django
- HTML, CSS
- SQLite (default Django DB)

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`

2. Apply migrations  
   `python manage.py migrate`

3. Run the server  
   `python manage.py runserver`

4. Open browser and go to  
   `http://127.0.0.1:8000`

## Database Concepts
- Relational data modeling
- CRUD operations
- Data validation and persistence
